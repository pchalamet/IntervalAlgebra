# Span Algebra

This is a simple F# library to compute intersections and unions on list of spans.

But, what is a span ? It is a value spanning on interval [start, end[

## Intersection

````
input1:
  [0, 1] = [0-1]
  [4, 5] = [4-5]
  [7, 10] = [7-10]
  [13, 15] = [13-15]
  [17, 20] = 17-20]

input2:
  [2, 3] = [2-3]
  [5, 6] = [5-6]
  [8, 11] = [8-11]
  [12, 14] = [12-14]
  [16, 21] = [16-21]

result:
  [8, 10] = [7-10] ∩ [8-11]
  [13, 14] = [12-14] ∩ [13-15]
  [17, 20] = [16-21] ∩ 17-20]
````

## Union

````
input1:
  [0, 1] = [0-1]
  [4, 5] = [4-5]
  [7, 10] = [7-10]
  [13, 15] = [13-15]
  [17, 20] = 17-20]

input2:
  [2, 3] = [2-3]
  [5, 6] = [5-6]
  [8, 11] = [8-11]
  [12, 14] = [12-14]
  [16, 21] = [16-21]

result:
  [0, 1] = [0-1]
  [2, 3] = [2-3]
  [4, 6] = [4-5] ∪ [5-6]
  [7, 11] = [7-10] ∪ [8-11]
  [12, 15] = [12-14] ∪ [13-15]
  [16, 21] = [16-21] ∪ 17-20]
````
